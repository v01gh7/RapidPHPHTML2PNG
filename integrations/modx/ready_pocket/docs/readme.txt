RapidHTML2PNG MODX Revo package
--------------------------------

This package installs:
- plugin: RapidHTML2PNGHeaderButton
- snippet: RapidHTML2PNGBulkRender
- system settings: rapidhtml2png_*
- files under assets/core/connectors paths

After install:
1) open MODX manager and clear cache
2) fill system settings rapidhtml2png_convert_url and rapidhtml2png_convert_api_key
3) reload manager and use header button "Отрендерить"
