### Project Overview
Building a PHP script to convert HTML blocks to PNG images with transparent background.
Key features:
- POST API accepts HTML blocks and CSS URL
- Content-based caching (MD5 hash)
- Auto-detects rendering libraries (wkhtmltoimage, ImageMagick, GD)
- Works on shared hosting (PHP 7.4+)
